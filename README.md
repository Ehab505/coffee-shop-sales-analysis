# ☕ Coffee Shop Sales Analysis

## Project Overview

This project analyzes **149,116 coffee shop transactions** from three New York City locations between **January and June 2023**. The objective was to identify the key factors influencing sales performance through data cleaning, statistical analysis, and interactive dashboard development using Microsoft Excel.

## Objectives

- Analyze sales performance across store locations.
- Compare weekday and weekend transaction values.
- Evaluate revenue differences among product categories.
- Analyze customer spending by time of day.
- Visualize business performance through an interactive dashboard.

## Dashboard Features

- KPI Cards
- Monthly Revenue Trend

- Revenue by Time of Day
- Interactive Slicers
- Dynamic Pivot Charts

## Key Findings

- Product category was the strongest predictor of transaction revenue, explaining **33.6%** of revenue variance.
- Morning transactions generated significantly higher average revenue than afternoon and evening transactions.
- Store location differences were statistically significant but had negligible practical impact.
- No significant difference was found between weekday and weekend average transaction values.
